# Level Plain Pro — Full Site (lpp-newcode)

Complete 19-page website. Flat structure — all files sit at the repo root.
Header and footer are written directly into every page (no JS injection),
so every page renders fully styled even if JavaScript is blocked.

## Deploy to GitHub Pages
1. Upload ALL files (19 .html + styles.css + script.js + .nojekyll) to the
   ROOT of your repo — drag the files, not the folder around them.
2. Settings → Pages → Deploy from branch → main, / (root).
3. Visit https://YOUR-USERNAME.github.io/REPO-NAME/

## Pages
index, services (+6 detail pages), process, results, about, contact,
resources (FAQ + glossary), blog (+3 posts), legal, 404.
